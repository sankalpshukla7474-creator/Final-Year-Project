# Endpoint Security Anomaly Detection System

This repository contains a comprehensive system designed to detect anomalies in endpoint security. The system is built to monitor and analyze various metrics related to endpoints such as network activity, system performance, and user behavior.

## Features

- **Real-time Monitoring**: Continuous monitoring of endpoint activities.
- **Anomaly Detection Algorithms**: Advanced algorithms to identify unusual patterns that could indicate security breaches.
- **Alert System**: Immediate alerts for detected anomalies.
- **Reporting**: Detailed reports on security incidents and trends over time.

## Getting Started

### Prerequisites
- Python 3.x
- Required libraries: `numpy`, `pandas`, `scikit-learn`

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/sankalpshukla7474-creator/Endpoint-Security-Anomaly-Detection-System.git
   cd Endpoint-Security-Anomaly-Detection-System
   ```
2. Install dependencies:
   ```bash
   pip install numpy pandas scikit-learn
   ```

### Running the System
1. Configure the system by editing `config.yaml`.
2. Start the monitoring process:
   ```bash
   python src/main.py
   ```
